## Welcome to the my java project

## Compiling the app

```bash
$ mvn compile
```

## Running the app

```bash
$ mvn exec:java -Dexec.mainClass="academy.kovalevskyi.codingbootcamp.week0.day2.Main" 
```

## Create jar file

```bash
$ mvn clean package
```

## Running jar file

```bash
$ java -jar ./target/bootcamp-1.0-SNAPSHOT.jar
```
